# react-wordpress-api-project
# پروژه اتصال ریکت به وردپرس
این پروژه به منظور نمایش توانایی اتصال **React** به **WordPress REST API** ساخته شده است.

 ویژگی‌ها:
- دریافت خودکار مقالات از وردپرس
- قابلیت جستجوی آنی در بین عناوین
- مدیریت وضعیت بارگذاری (Loading State)
- طراحی کاملاً ریسپانسیو با CSS
